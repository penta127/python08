# py08

プロジェクト構成や依存関係、設定ファイルを扱う演習フォルダです。
実際の Python プロジェクトの形に近い構造を体験します。

## どんなことをやるか

- `ex0/construct.py`: 設計図的なスクリプトの例
- `ex01/`: `pyproject.toml` や `requirements.txt` を使ったパッケージ構成
- `ex02/`: `.env` 形式の設定ファイルと環境変数の使い方

## 目的

- Python プロジェクトの構造を理解する
- 依存関係の管理や設定ファイルの扱い方を学ぶ
- 実務に近いプロジェクトの準備手順を確認する

## 実行方法

1. `cd py08/ex0` / `cd py08/ex01` / `cd py08/ex02`
2. `python3 *.py`

`py08` は、Python アプリを構造的に作る感覚を養うステージです。
