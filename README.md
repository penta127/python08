# py08

このフォルダでは、Python プロジェクトの構成や依存関係、設定ファイルの扱い方を学びます。
実際の開発に近い形で、モジュール化や環境設定を体験します。

## 何をする課題か

- `ex0/construct.py`: ひとつの処理を部品としてまとめて再利用しやすくする
- `ex01/`: `pyproject.toml` や `requirements.txt` で必要なライブラリを管理する
- `ex02/`: `.env` などの設定ファイルを使って、実行環境を整える

## ここで学ぶこと

- Python プロジェクトの構造を理解する
- 依存関係の管理や設定ファイルの扱い方を学ぶ
- 実務に近いプロジェクトの準備手順を確認する

## 実行方法

1. `cd py08/ex0` / `cd py08/ex01` / `cd py08/ex02`
2. `python3 *.py`

`py08` は、Python アプリを構造的に作る感覚を養うステージです。
